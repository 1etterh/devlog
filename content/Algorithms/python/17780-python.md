[백준 17780번](https://www.acmicpc.net/problem/17780)

정답률이 50%대라서 쉬운 문제라고 생각했는데 오산이었다..ㅜ
실제로 풀어보니 굉장히 까다로운 문제였음..
사실 내가 푼 방법이 틀릴 수도 있긴 한데 잊어버리기 전에 풀이 과정을 작성해봄.


#### 입력
```
n,k # n는 체스판의 크기, k는 말의 갯수
for _ in range(n):
	체스판 정보(row)
for _ in range(k):
	말의 정보(row, column, direction)
```

#### 풀이과정

함수
1. isGround(x)함수: 
	움직이려는 말이 0층에 있는지 확인
2. canMove(x)함수:
	움직이려는 말이 실제로 움직일 수 있는지 확인
3. backdir(x):
	말을 움직일 수 없는 경우 방향을 반대로 바꿔줌
4. move(x):
	말을 이동.

변수
1. status board 
	1. 말이 있는경우: 스택(horses)
	2. 말이 없는경우: -1

2. nextTurn(Boolean)
	1. 게임이 끝났는지 확인


풀이 과정에서는 정말 길다고 느꼈는데 풀고 난 후 다시 보니 100줄도 안돼서 놀라웠다...
#### 코드
```
import sys

n,k=map(int, sys.stdin.readline().split())

board=[]

horse =[]

rDir = [0,0,-1,1]

cDir = [1,-1,0,0]

status=[[-1]*n for _ in range(n)]

for _ in range(n):
board.append(list(map(int,sys.stdin.readline().split())))

for i in range(k):
    r,c,d=map(int, sys.stdin.readline().split())
    horse.append([r-1,c-1,d-1])#편의상 0부터 시작
    status[r-1][c-1]=[i] #status에 스택 추가

def canMove(x):
    r,c,d=horse[x]
    nr, nc = r+rDir[d], c+cDir[d]

    if 0<=nr<n and 0<=nc<n:
        if board[nr][nc]!=2:
            return True
        else:
            return False
    return False

def isGround(x):

    r,c,d=horse[x]

    if status[r][c][0]==x:

        return True

    else:

        return False

def backdir(x):

    if horse[x][2]%2==0:

        horse[x][2]+=1

        return

    else:

        horse[x][2]-=1

        return

def move(x):
    r,c,d=horse[x]    
    nr, nc = r+rDir[d], c+cDir[d]
    if board[nr][nc]==1:

        status[r][c].reverse()

    if status[nr][nc]==-1:

        status[nr][nc]=status[r][c]

    else:

        status[nr][nc].extend(status[r][c])

    for s in status[nr][nc]:

        horse[s][:2]=[nr,nc]

    status[r][c]=-1

    if len(status[nr][nc])>3:

        global nextTurn

        nextTurn=False

cnt=0

nextTurn=True

while nextTurn:

    cnt+=1

    if cnt==1000:

        nextTurn=False

    for m in range(k):

        if isGround(m):

            if canMove(m):

                move(m)

            else:

                backdir(m)

                if canMove(m):

                    move(m)

                else:

                    backdir(m)

  

if cnt!=1000:

    print(cnt)

else:

    print(-1)
```

사실 절대로 끝나지 않는 경우를 계산할 방법이 떠오르지 않아 그냥 1000을 넘으면 -1을 출력하도록 했더니 맞긴 했다..
다시 풀어볼 때 다시 생각해봐야겠당
