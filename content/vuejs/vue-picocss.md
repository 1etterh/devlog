#### Picocss
초경량 css 프레임워크
기본적인 HTML 태그 디자인을 이쁘게 만들어준다.
###### Git 주소
```
https://github.com/paulo-pertierra/vue-picocss
```

###### 사용법
1. vue 프로젝트 폴더에서 설치 명령어 실행
```
npm install vue-picocss
```

2. vue 프로젝트의 main.js에 다음 코드 추가
```
import "vue-picocss/css/pico.min.css"
```
